# SEO Fundamentals

Apply on-page SEO best practices to web content, ensuring pages are discoverable, crawlable, and rank well in search engines.

## On-Page SEO Essentials

### Title Tags
- Keep under 60 characters so they display fully in SERPs.
- Place the primary keyword near the beginning.
- Make each page title unique and descriptive.

### Meta Descriptions
- Write compelling summaries under 155 characters.
- Include the primary keyword naturally.
- Add a call to action when appropriate ("Learn how...", "Discover...").
- Meta descriptions do not directly affect ranking but influence click-through rate.

### Heading Structure
- Use a single `<h1>` per page that contains the primary keyword.
- Organize content with `<h2>` and `<h3>` sub-headings for scannability.
- Headings should form a logical outline of the page content.

### URL Structure
- Use short, descriptive, hyphenated slugs (`/seo-fundamentals` not `/page?id=42`).
- Include the target keyword in the URL path.
- Avoid unnecessary nesting and parameters.

## Structured Data

- Add JSON-LD schema markup for articles, products, FAQs, breadcrumbs, and organization info.
- Validate with Google's Rich Results Test before deployment.
- Structured data enables rich snippets (star ratings, FAQ dropdowns, breadcrumbs) in search results.

## Site Speed

- Target a Largest Contentful Paint (LCP) under 2.5 seconds.
- Compress images (WebP/AVIF), lazy-load below-the-fold assets.
- Minimize render-blocking CSS and JavaScript.
- Use a CDN for static assets.
- Monitor Core Web Vitals (LCP, FID/INP, CLS) regularly.

## Mobile-First Indexing

- Google indexes the mobile version of pages first.
- Ensure responsive design with no horizontal scrolling.
- Touch targets should be at least 48x48 CSS pixels.
- Verify mobile usability in Google Search Console.
- Avoid intrusive interstitials that block content on mobile.

## Internal Linking

- Link related pages with descriptive anchor text (not "click here").
- Maintain a shallow site architecture -- important pages within 3 clicks of the homepage.
- Use breadcrumbs for navigation and structured data.

## Content Quality Signals

- Write for the user first, search engines second.
- Satisfy search intent: informational, navigational, transactional, or commercial.
- Keep content up to date and indicate the last-updated date.
- Use original images and media where possible.
